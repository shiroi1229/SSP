# SSP Roadmap: v3.0 to v4.0

This document outlines the high-level strategic roadmap for the Shiroi System Platform (SSP) evolution, from the Meta-Contract System (v3.0) to Akashic Integration (v4.0).

| Phase | Code Name                    | Goal                                           | Status     |
|-------|------------------------------|------------------------------------------------|------------|
| v3.0  | Meta-Contract System         | 契約の自動生成・動的接続・自己拡張             | 構想開始   |
| v3.1  | Cognitive Graph Engine       | 全モジュール・概念・文脈の意味ネットワーク化   | 設計準備中 |
| v3.2  | Self-Reasoning Loop          | Reasoning／Planning／Memoryを統合した自己思考層 | 研究段階   |
| v3.3  | Distributed Persona Fabric   | 複数AI人格（サブシロイ）間の協調・競合学習     | 計画中     |
| v3.4  | Collective Intelligence Core | 群知能的最適化（マルチユーザー・マルチAI連携） | 構想段階   |
| v3.5  | Evolution Mirror             | 進化過程の自己観測・再帰的評価（自己監査AI）   | 未着手     |
| v4.0  | Akashic Integration          | SSP全履歴を統合する“記憶宇宙（Akashic DB）”の実装 | 長期目標   |
